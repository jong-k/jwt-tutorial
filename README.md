## 프로젝트 환경 및 실행 방법
### node version 확인
node 16.18.0
### 패키지 설치
`yarn install`
### 프로젝트 실행
`yarn start`

## 프로젝트 상세
### 사용 라이브러리
- react-router-dom (라우팅 관련)
- react-icons (아이콘 활용)
- classnames (className 동적 생성)
- react-virtualized (렌더링 성능 최적화)
- styled-components (스타일링)
